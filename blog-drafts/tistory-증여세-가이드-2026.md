## 본문

### 증여세란?

증여세는 타인으로부터 재산을 무상으로 받았을 때 내는 세금입니다. 현금, 부동산, 주식, 채무 면제 등 모든 경제적 이익이 증여에 해당합니다.

---

### 2026년 증여세 면제 한도 (증여재산공제)

| 증여자 → 수증자 관계 | 공제 한도 | 기간 |
|---|---|---|
| 배우자 → 배우자 | **6억원** | 10년 합산 |
| 직계존속(부모) → 성인 자녀 | **5,000만원** | 10년 합산 |
| 직계존속(부모) → 미성년 자녀 | **2,000만원** | 10년 합산 |
| 직계비속(자녀) → 부모 | **5,000만원** | 10년 합산 |
| 기타 친족(형제·삼촌 등) | **1,000만원** | 10년 합산 |

핵심 포인트:
- 10년 단위로 합산됩니다. 10년 전에 증여한 금액은 리셋.
- 여러 사람에게 받아도 **같은 관계**면 합산합니다. (아빠 3,000만+엄마 3,000만 = 합산 6,000만 → 공제 5,000만 초과)

---

### 증여세 세율표 (누진세율)

| 과세표준 | 세율 | 누진공제 |
|---|---|---|
| 1억원 이하 | 10% | - |
| 1억~5억원 | 20% | 1,000만원 |
| 5억~10억원 | 30% | 6,000만원 |
| 10억~30억원 | 40% | 1억6,000만원 |
| 30억원 초과 | 50% | 4억6,000만원 |

---

### 증여세 계산 예시

**부모 → 성인 자녀에게 1억원 현금 증여 시**

```
증여재산: 1억원
공제: 5,000만원 (직계존속)
과세표준: 5,000만원
세율: 10% (1억 이하)
증여세: 500만원
```

**부모 → 미성년 자녀에게 5,000만원 증여 시**

```
증여재산: 5,000만원
공제: 2,000만원 (미성년)
과세표준: 3,000만원
세율: 10%
증여세: 300만원
```

### 증여세 절세 전략

1. **10년 주기 분할 증여**: 공제 한도가 10년마다 리셋되므로, 자녀가 어릴 때부터 10년마다 증여하면 세금 없이 큰 금액 이전 가능
2. **부부 공동 증여**: 아빠와 엄마 각각 5,000만원씩 증여하는 건 합산됨. 대신 할아버지·할머니도 활용하면 추가 공제 가능
3. **시가 낮을 때 부동산 증여**: 부동산 가격이 낮은 시점에 증여하면 평가액이 낮아져 세금 절감
4. **자녀 명의 적금·펀드 활용**: 증여 후 자녀 명의로 투자하면 이후 수익은 자녀 소득

### 증여세 신고 기한

- 증여받은 날이 속하는 달의 말일로부터 **3개월 이내** 신고
- 기한 내 신고 시 **3% 신고세액공제** 적용
- 미신고 시 무신고가산세(20%) + 납부지연가산세 부과
