## 본문

### 장기보유특별공제란?

장기보유특별공제(장특공제)는 부동산을 오래 보유할수록 양도소득세를 깎아주는 제도입니다. 보유기간이 길수록 공제율이 올라가서 세금 부담이 크게 줄어듭니다.

---

### 1세대 1주택 장특공제율 (거주기간 포함)

1세대 1주택으로 양도차익이 12억원을 초과하는 경우, 초과분에 대해 아래 공제율이 적용됩니다.

**보유기간 공제 (연 4%)**

| 보유기간 | 공제율 |
|---|---|
| 3년 | 12% |
| 4년 | 16% |
| 5년 | 20% |
| 6년 | 24% |
| 7년 | 28% |
| 8년 | 32% |
| 9년 | 36% |
| 10년 이상 | 40% |

**거주기간 공제 (연 4%)**

| 거주기간 | 공제율 |
|---|---|
| 2년 | 8% |
| 3년 | 12% |
| 4년 | 16% |
| 5년 | 20% |
| 6년 | 24% |
| 7년 | 28% |
| 8년 | 32% |
| 9년 | 36% |
| 10년 이상 | 40% |

→ 보유 10년 + 거주 10년이면 **최대 80% 공제** 가능.

---

### 일반 부동산 장특공제율 (1주택 비과세 비해당)

다주택자, 비주거용 부동산(토지·상가), 1주택이지만 비과세 요건 미충족 시 적용:

| 보유기간 | 공제율 |
|---|---|
| 3년 | 6% |
| 4년 | 8% |
| 5년 | 10% |
| 6년 | 12% |
| 7년 | 14% |
| 8년 | 16% |
| 9년 | 18% |
| 10년 | 20% |
| 11년 | 22% |
| 12년 | 24% |
| 13년 | 26% |
| 14년 | 28% |
| 15년 이상 | 30% |

→ 일반 부동산은 최대 **30%** 공제.

---

### 장특공제 계산 예시

**1주택자 (보유 8년, 거주 5년, 양도차익 15억원)**

```
비과세 한도: 12억원
과세 대상 차익: 15억 - 12억 = 3억원 (비율 안분)
장특공제율: 보유 32% + 거주 20% = 52%
공제액: 과세 대상 양도차익 × 52%
→ 실제 과세표준이 크게 줄어듦
```

**다주택자 (보유 10년, 양도차익 2억원)**

```
장특공제율: 20% (일반 부동산 기준)
공제액: 2억 × 20% = 4,000만원
과세표준: 2억 - 4,000만 = 1.6억원
```

### 주의사항

- **다주택자 중과 시**: 조정대상지역 다주택자는 장특공제 적용이 **배제**될 수 있음 (중과 유예 기간 확인 필요)
- **보유기간 기산점**: 취득일(잔금일 또는 등기접수일 중 빠른 날)부터 양도일까지
- **거주기간 증빙**: 전입신고 기준. 실거주 증빙이 필요할 수 있음
- **상속·증여 취득**: 보유기간 계산 시 피상속인·증여자의 보유기간 승계 여부 확인
