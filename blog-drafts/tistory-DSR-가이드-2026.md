## 본문

### DSR이란?

DSR(총부채원리금상환비율)은 연소득 대비 모든 대출의 연간 원리금 상환액 비율입니다.

```
DSR = (모든 대출의 연간 원리금 상환액 ÷ 연소득) × 100
```

2026년 현재 **DSR 40%** 규제가 적용됩니다. 쉽게 말해, 연봉의 40%까지만 대출 상환에 쓸 수 있다는 뜻입니다.

### 스트레스 DSR이란?

2024년 하반기부터 단계적으로 도입된 스트레스 DSR은 **미래 금리 상승 가능성을 미리 반영**해서 대출 한도를 계산하는 방식입니다.

기존 DSR은 현재 대출 금리로만 계산했지만, 스트레스 DSR은 **가산금리(스트레스 금리)**를 더해서 계산합니다.

```
스트레스 DSR = 현재 금리 + 스트레스 가산금리로 계산한 원리금 기준
```

→ 같은 연봉·같은 금리여도 **대출 한도가 줄어듭니다**.

---

### 스트레스 DSR 적용 단계 (2026년 기준)

| 단계 | 시기 | 스트레스 금리 반영 비율 |
|---|---|---|
| 1단계 | 2024.2~ | 25% |
| 2단계 | 2024.9~ | 50% |
| 3단계 | 2025.7~ | 100% (전면 적용) |

2026년 현재 **3단계(100%)가 적용** 중이며, 스트레스 금리는 한국은행 기준으로 산정됩니다.

※ 스트레스 DSR 적용 시 위 한도에서 5~15% 감소할 수 있습니다.
※ 기존 대출(신용대출, 카드론 등)이 있으면 한도가 줄어듭니다.

---

### 기존 대출이 있을 때 계산법

예시: 연소득 6,000만원, 기존 신용대출 월 상환 30만원

```
연간 상환 가능 총액 = 6,000만 × 40% = 2,400만원
기존 대출 연 상환 = 30만 × 12 = 360만원
신규 대출 상환 가능 = 2,400 - 360 = 2,040만원
→ 신규 주담대 한도: 약 3.3억원 (30년/4.5% 기준)
```

기존 대출을 먼저 갚으면 그만큼 한도가 올라갑니다.

### DSR 줄이는 방법

1. **대출 기간 늘리기**: 20년 → 30년으로 늘리면 월 상환액이 줄어 DSR 비율 하락
2. **기존 대출 상환**: 신용대출·카드론 먼저 갚기
3. **소득 증빙 강화**: 부부합산 소득, 부업 소득 추가 증빙
4. **거치기간 활용**: 일정 기간 이자만 내는 방식 (단, DSR 산정 시 만기 일시상환 기준으로 잡힐 수 있음)
