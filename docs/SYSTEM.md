# Auto-Factory: AI 자가발전 수익 시스템

## 시스템 목적

AI가 웹 계산기/도구를 자동으로 기획·생산·배포하여 애드센스 광고 수익을 창출하는 **자가발전 시스템**.

- 사람의 개입 없이 콘텐츠를 지속적으로 생산
- 검색 트래픽을 유입할 수 있는 실용적인 웹 도구(계산기) 제작
- 애드센스 광고를 통해 수익 자동화

---

## 12단계 프로세스

| 단계 | 이름 | 설명 |
|------|------|------|
| 1 | **기획** | 검색 수요가 높은 계산기 주제 선정 (키워드 조사) |
| 2 | **생산** | 템플릿 기반으로 HTML/CSS/JS 계산기 페이지 생성 |
| 3 | **검증** | 계산 로직 정확성 및 UI/UX 동작 검증 |
| 4 | **수정** | 검증에서 발견된 오류 및 개선사항 수정 |
| 5 | **재검증** | 수정 후 재검증 (통과 시 다음 단계 진행) |
| 6 | **적용** | 웹서버/호스팅에 파일 배포 |
| 7 | **검사** | 배포 후 실제 환경에서 동작 확인 (SEO, 광고, 반응형) |
| 8 | **수정** | 배포 환경에서 발견된 문제 수정 |
| 9 | **재적용** | 수정된 버전 재배포 |
| 10 | **검사** | 최종 품질 검사 및 구글 색인 요청 |
| 11 | **점검** | 주기적 성과 모니터링 (트래픽, 수익, 순위) |
| 12 | **재설계** | 성과 데이터 기반으로 다음 주제 선정 및 템플릿 개선 → 1단계로 |

## 폴더 구조

```
auto-factory/
├── pipeline/       # 자동화 스크립트 (생성, 배포, 모니터링)
├── products/       # 생산된 웹 계산기/도구 (배포 준비 완료 파일)
├── templates/      # 계산기 HTML 템플릿 (재사용 가능한 기본 틀)
├── monitoring/     # 모니터링 데이터 (트래픽, 수익, 색인 현황)
└── docs/           # 시스템 문서
    └── SYSTEM.md   # 이 파일
```

### 각 폴더 상세

- **pipeline/**: Python/Shell 스크립트로 생산 파이프라인 자동화
- **products/**: 완성된 계산기별 서브폴더 (`/products/퇴직금계산기/index.html` 등)
- **templates/**: 재사용 가능한 기본 HTML 틀 (카테고리별 분류 예정)
- **monitoring/**: CSV/JSON 형태의 성과 데이터 저장
- **docs/**: 운영 가이드, 프로세스 문서

---

## 수익 모델

1. **구글 애드센스**: 각 계산기 페이지에 광고 삽입
2. **자연 검색 트래픽**: SEO 최적화된 롱테일 키워드 타겟
3. **복리 성장**: 페이지 수 증가 → 트래픽 증가 → 수익 증가 → 재투자
