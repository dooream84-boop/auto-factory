# CocoFlow 수익 채널 다변화 보고서
작성일: 2026-04-18
목적: AdSense 단일 수익 구조의 한계(현재 0 클릭 → 수익 0)를 극복하기 위한 제휴·어필리에이트 채널 리서치. coco가 직접 가입 의사결정을 내릴 수 있도록 카테고리별 제휴사와 진입 조건을 정리함.

---

## 수익 믹스 전략 (요약)

| 레이어 | 수익원 | 매출 유형 | 예상 전환율 | 단가 |
|---|---|---|---|---|
| 1 | Google AdSense | CPC/CPM | - | 노출당 낮음 |
| 2 | **대출·금융 제휴** | CPA | 0.5~2% | 건당 5,000~30,000원 |
| 3 | **세금 환급 제휴** | CPA | 1~3% | 건당 10,000~40,000원 |
| 4 | **노무 상담 제휴** | CPA/리드 | 0.3~1% | 건당 5,000~15,000원 |
| 5 | **자동차 보험 제휴** | CPA | 0.5~2% | 건당 10,000~50,000원 |

**핵심 원리**: 계산 결과를 본 사용자는 "구체적인 다음 행동"(대출 실행, 세금 신고, 보험 가입)을 앞두고 있다. AdSense보다 이 의도와 직결된 서비스가 압도적으로 전환율이 높다.

## 1. 대출·금융 비교 제휴

### 1-1. 토스(비바리퍼블리카) — 최우선 추천
**연관 계산기**: dsr-calculator, loan-calculator, compound-interest-calculator
- 공식: [토스 제휴 파트너](https://toss.im) — 금융 비교 서비스 강자
- **가입 방법**: 토스 자체 제휴 프로그램은 직접 모집 대상이 제한적. 대부분 중개 어필리에이트 플랫폼(디비디비딥·애드픽)을 통해 연결
- **수수료**: 대출 신청 완료 1건당 5,000~15,000원 추정 (상품별)
- **장점**: 국내 금융 앱 중 인지도 1위, 전환율 안정적
- **진입 난이도**: 중 (플랫폼 경유)

### 1-2. 핀다(Finda) — 개인신용대출 비교 플랫폼
**연관 계산기**: loan-calculator, dsr-calculator
- 공식: [finda.co.kr](https://finda.co.kr)
- **가입 방법**: 핀다 자체 어필리에이트 프로그램 있음. 가입 폼 제출 → 심사 → 승인 시 고유 트래킹 링크 발급
- **수수료**: 대출 신청 1건당 약 10,000~25,000원 (상품 등급별 차등)
- **장점**: 핀다 UI로 바로 연결되므로 CocoFlow가 따로 비교 DB를 구축할 필요 없음
- **진입 난이도**: 중~상 (콘텐츠 품질·트래픽 요구)

### 1-3. 뱅크샐러드
**연관 계산기**: 금융 전반 (savings, compound-interest 등)
- 공식: [banksalad.com](https://banksalad.com)
- **가입 방법**: 파트너십 문의(제휴 담당자 메일) → 조건 협의
- **수수료**: 협의형 (가입 기반)
- **장점**: 재테크 전반의 강력한 브랜드
- **진입 난이도**: 높음 (직접 계약)

### 1-4. 디비디비딥(DB DB Deep) — **첫 진입용 1순위**
**연관 계산기**: 전 금융/세무 카테고리
- 공식: [dbdbdeep.com](https://dbdbdeep.com)
- **가입 방법**: 웹사이트 어필리에이트 가입 폼 → 사이트 심사 → 승인 시 광고주 다수 선택 가능
- **광고주**: 대출 비교, 신용카드, 자동차보험, 이사업체 등 수십 종
- **수수료**: 광고주별 상이 (3,000~30,000원)
- **장점**: 여러 광고주를 한 번에 연결 가능. **사이트 심사 기준이 상대적으로 낮음**
- **진입 난이도**: 낮음 (가장 먼저 시도 권장)

### 1-5. 애드픽(Adpick)
- 공식: [adpick.co.kr](https://adpick.co.kr)
- CPA 위주, 앱 설치·가입·구매 등 다양한 오퍼
- **진입 난이도**: 낮음

## 2. 세금 환급 제휴

### 2-1. 삼쩜삼(3.3) — **세금 환급 시장 1위**
**연관 계산기**: tax-refund-calculator, income-tax-calculator, financial-income-tax-calculator
- 공식: [3o3.co.kr](https://3o3.co.kr) — 자비스앤빌런즈 운영
- **가입 방법**: 삼쩜삼 공식 어필리에이트는 제한적. 주로 디비디비딥·애드픽 경유, 혹은 제휴 담당자 직접 문의 (contact@jobis.co)
- **수수료**: 가입 완료 1건당 약 10,000~40,000원 추정
- **타겟 고객**: 프리랜서, 아르바이트생, 과세표준 낮은 근로자 → 종합소득세/연말정산 계산기 이용자와 완벽 일치
- **진입 난이도**: 중

### 2-2. 택스테크(자비스) — 사업자용 세무 대행
**연관 계산기**: vat-calculator, income-tax-calculator
- 공식: [jobis.co](https://jobis.co)
- 소상공인·사업자 대상 → 부가세·종합소득세 계산기 이용자와 겹침
- **진입 난이도**: 중

### 2-3. 세무사 매칭 플랫폼 (숨고, 크몽)
**연관 계산기**: gift-tax, inheritance-tax, capital-gains-tax, acquisition-tax, overseas-stock-tax
- **가입 방법**: 숨고/크몽 자체 어필리에이트 프로그램 (있다면)
- 고부가 세무 영역은 전환 단가가 높음 (상담료 5만~30만원 → CPA 리드 단가도 높음)
- **진입 난이도**: 중

## 3. 노무 상담 제휴

### 3-1. 노무법인·노무사 사무소 직접 제휴
**연관 계산기**: severance, salary, annual-leave, midterm-leave, insurance
- 개별 노무사 사무소에 직접 제휴 제안 (cold outreach)
- 리드 전달 1건당 5,000~20,000원 협상 가능
- **진입 난이도**: 중~높음 (개별 영업)

### 3-2. 노무 상담 플랫폼 (노무114, 고용노동통합포털 등)
- 플랫폼 자체 어필리에이트는 드묾
- 고용노동부 산하 공공 서비스는 광고 연계 불가

### 3-3. 숨고·크몽 (노무사 카테고리)
- 숨고는 "서비스 신청 리드" 기반 수수료 체계 있음
- 수수료는 서비스 카테고리별 상이

## 7. 제휴 슬롯 연동 방법 (기술 가이드)

모든 계산기 페이지에 이미 제휴 슬롯 placeholder가 삽입되어 있음:

```html
<div class="affiliate-slot" data-category="loan" style="display:none;">
  <!-- 여기에 제휴 카드가 들어감 -->
</div>
```

`data-category` 값에 따라 적절한 제휴 서비스 카드를 JS로 주입하면 됨.

**카테고리 매핑 요약:**

| data-category | 매칭 계산기 | 후보 제휴사 |
|---|---|---|
| loan | dsr, loan | 토스/핀다/디비디비딥 대출 |
| savings | compound-interest, savings | 예적금 비교, 뱅크샐러드 |
| tax | income-tax, tax-refund, financial-income-tax, overseas-stock-tax, capital-gains-tax, gift-tax, inheritance-tax, vat | 삼쩜삼, 자비스, 세무사 매칭 |
| realestate | acquisition-tax | 세무사 매칭, 부동산 관련 대출 |
| auto | automobile-tax | 자동차보험 비교 |
| labor | severance, salary, hourly-wage, annual-leave, midterm-leave, insurance | 노무사 매칭, 구인구직(잡코리아 등) |
| maternity | due-date | 태아보험·실손보험 |
| health | bmi, calorie | 건강검진·실손보험 |
| utility | electricity | (현재 적합 제휴 없음. 광고만) |
| none | dday, korean-age | 제휴 없음 |

**활성화 예시 (제휴 가입 후):**
```javascript
// 공통 affiliate.js — 모든 페이지에 include
document.querySelectorAll('.affiliate-slot[data-category="loan"]').forEach(function(slot){
  slot.innerHTML = `
    <div class="affiliate-card" style="background:#fffbeb;border:1px solid #fde68a;border-radius:10px;padding:16px;margin:20px 0;">
      <p style="font-size:0.78rem;color:#92400e;margin-bottom:8px;">💡 광고 · 제휴 링크</p>
      <p style="font-weight:700;margin-bottom:6px;">내 DSR 기준 최저금리 찾기</p>
      <p style="font-size:0.86rem;color:#475569;margin-bottom:10px;">핀다에서 48개 은행을 한 번에 비교</p>
      <a href="TRACKING_URL" target="_blank" rel="sponsored noopener" style="...">무료 비교하기 →</a>
    </div>`;
  slot.style.display = 'block';
});
```
